# Job Application Tracker 
**Version:** 1.0
---
## 📖 Description

The **Job Application Tracker** is a command-line application written in Python that helps users organize and manage their job applications. It allows users to add new applications, update their status, search for applications, delete entries, and view statistics, while storing all data in a JSON file.
---
This project was built to practice Python programming, object-oriented design, JSON file handling, and Git version control.
---

## ✨ Features

* Add new job applications

  * Company name
  * Position
  * Location
  * Application date
  * Current status
* View all saved applications
* Search applications by company
* Update application status
* Delete applications
* View the total number of applications
* View statistics by application status

  * Applied
  * Interview
  * Accepted
  * Rejected
* Input validation for user entries
* Automatic data storage using JSON
